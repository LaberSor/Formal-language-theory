Группа АВТ-813
Основные функции
Файл

  • Создать

    - создание нового пустого файла с выбранным именем в выбранной дериктории.

  • Открыть

    - открытие существующего файла.

  • Сохранить

    - сохранить все изменения.

  • Сохранить как

    - сохранить файл под выбранным именем в выбранной дериктории.

  • Выход

    - выход из текущего файла.
Правка

  • Отменить

    - отмена последнего действия.

  • Повторить

    - повторяет отменненое действие.

  • Вырезать

    - помещает выделенный фрагмент в буфер обмена, и одновременно удаляет его из файла.

  • Копировать

    - помещает фрагмент в буфер обмена, оставляя файл без изменений.

  • Вставить

    - вставляет в текущий документ фрагмент, находящийся в буфере обмена.

  • Удалить

    - удаляет выбранный фрагмент из файла.

  • Выделить все

    - выделяет весь текст в файле.

  • Удалить все

    - удаляет весь текст в окне.
Текст
  • В разработке
Обработка РВ
  • Определяет адреса в тексте. Ниже указаны условия определения.
    Регулярное выражение: ((Протокол)?(Префикс)?[Имя]+.\b(\.){1}(Домен){1}(/Маршрут)*
    • Протокол: ((https?|ftp):\/\/) - определяет наличие протоколов http, https, ftp
    • Префикс: (www\.) - определяет наличие префикса www
    • Имя: [a-zA-Z0-9\-\.\~\\_] - определяет имя сайта из допустимых символов
    • Домен: (ru|com|org|net|mil|edu|COM|ORG|NET|MIL|EDU) - определяет домен: ru, com, org, net, mil, edu
    • Маршрут: (\/[a-zA-Z0-9\-\.]+) - определяет наличие маршрута
Обработка ДКА

    • Маршрут: (\/[a-zA-Z0-9\-\.]+) - определяет наличие маршрута
